# Inboxo — Website

Marketing site for **Inboxo**, a done-for-you email marketing agency for e-commerce brands
(Figma email design, Klaviyo flows, campaigns, ad copy). One goal: growing client revenue.

- Single static page — `index.html` + `assets/`. No build step.
- Motion: GSAP 3.13 + ScrollTrigger and a Three.js r160 scroll-driven 3D intro (CDN, progressive enhancement).
- Deployed on Vercel.

## Local preview

Any static server works, e.g.:

```bash
npx serve .
```

Then open the printed URL.
